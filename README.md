# job4j_url_shortcut

[![Java CI with Maven](https://github.com/Kirprox/job4j_url_shortcut/actions/workflows/maven.yml/badge.svg)](https://github.com/Kirprox/job4j_url_shortcut/actions/workflows/maven.yml)

## Описание проекта

job4j_url_shortcut — это REST-сервис для сокращения URL-адресов, предназначенный для использования сайтами-клиентами.

Сервис позволяет:
- регистрировать сайты
- авторизовываться с помощью JWT
- преобразовывать длинные URL в короткие коды
- выполнять редирект по короткой ссылке
- собирать статистику переходов


---

## Используемые технологии

- Java
- Spring Boot 2
- Spring Web
- Spring Security (JWT)
- Maven
- PostgreSQL
- Liquibase
- Swagger
